    This section contains the Brown Norway rat genome (Rattus norvegicus) from Ensembl. Researchers from the RGSC combined sequence and map resources in a bacterial artificial chromosome (BAC) with whole genome shotgun sequencing to assemble the genome. Developments in rat genomics look promising due to the support of bioinformatics and combined efforts of groups worldwide.

    Any data here relates to the November 2004 and July 2014 update of the Rat Genome Sequencing Consortium (RGSC) ran through our alternative pipeline.

    About the pipeline

    Flowchart image of the alternative pipeline
    (Pertea et al.,2016)

    The “new Tuxedo” protocol. This pipeline included HISAT2, StingTie and Ballgown that provided a complete analysis package for RNA-seq data and are open-source software tools. HISAT2 mapped the reads to the genome before StringTie assembled and merged the alignments into full length transcripts. Gffcompare was used next to compare genes and transcripts with the annotation and the report's statistics. StringTie then estimated abundances and created a new transcript table for input to Ballgown. R and Ballgown were then coherently used in the analysis of the differentially expressed genes and transcripts.
